◆ validerFormatNom()

bool util::validerFormatNom ( const std::string &  p_nom)

Méthode qui vérifie si une string correspond à un nom valide.

Un nom valide est constituer que de lettre et n'est pas constituer de deux espaces ou tirets de suite. ValiderFormatNon fonctionne de manière très simple. Il assume que le nom est valide, puis il passe à travers tout les caractères. En fesant cela, il y a trois possibilité. (1) Le caractères est une lettre. Si c'est vrai, on ne teste rien d'autre et on passe au suivant. (2) Le caractères est un espace ou un tiret. Dans ce cas, le code vérifie que le caractère suivant est un autre espace ou un tiret. Si ce n'est pas le cas on passe au caractère suivant sinon on considère le nom invalide. (3) Ce n'est pas un caractère valide donc on sort de l'ittération
